# 💼 Portfólio - Rubens dos Santos

Este é um portfólio profissional desenvolvido como Single Page Application (SPA) utilizando **HTML5, CSS3 e JavaScript puro**, sem frameworks.

O objetivo do projeto é apresentar habilidades técnicas, trajetória profissional e projetos de forma moderna, responsiva e com forte identidade visual.
